About me

I learned Python to program Software Defined Network controllers. I loved coding so much, I decided to become a software engineer late last December. So I spent January getting used to data structures and algorithms.

I had a wonderful experience being interviewed by Google in Feb- I solved two interesting problems and both the interviewers across the table encouraged me to continue what I was doing. I met awesome people at Hackathons in March who told me to start building stuff and I (am just getting started) haven't stopped since.

I used python to write the backend for a website to find travel buddies- learned SQL and how to use the webapp2 framework to deploy on the Google App Engine and showed it to my friends. Instead of arguing with them that I had built a Minimum Viable Product, I took the opportunity to learn some CSS, failed to make it usable on mobile phones and later patched it up to make it responsive across all devices- one week well spent.

I did not get into Google and there's a lot more to learn but I will soon get into an awesome company doing what I love- coding.
